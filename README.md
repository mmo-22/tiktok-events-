# 🎯 TikTok Live Reader

قارئ chat للبث المباشر على TikTok — يعمل فعلاً مع أي بث مباشر.

## المتطلبات
- Node.js 16+

## التشغيل

```bash
# تثبيت الحزم
npm install

# تشغيل السيرفر
npm start
```

ثم افتح المتصفح على: **http://localhost:3000**

## كيفية الاستخدام
1. شغّل السيرفر
2. افتح http://localhost:3000
3. أدخل يوزرنيم حساب TikTok يبث الآن
4. اضغط "اتصل"

## الميزات
- 💬 Chat مباشر في الوقت الفعلي
- 🎁 عرض الهدايا
- 👁️ عدد المشاهدين
- ❤️ الإعجابات والمتابعات
- 🔗 المشاركات
- تمييز المشرفين والمشتركين

## ملاحظة
يعتمد المشروع على مكتبة `tiktok-live-connector` مفتوحة المصدر.
قد تتغير TikTok الـ API من وقت لآخر.
